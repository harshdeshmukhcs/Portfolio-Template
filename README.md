# Frontend Portfolio Template

A modern and responsive frontend portfolio template built with a fast and lightweight stack. Designed to showcase projects, skills and experience and easy to customize for personal or professional use.

## Tech Stack
- React
- TypeScript
- Vite
- Tailwind CSS

## Features
- Responsive layout
- Clean and minimal UI
- Component-based architecture
- Fast development and production builds
- Easy customization and extension
